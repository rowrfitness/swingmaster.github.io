Don't track a history of commits of this website! No need to bloat this repository with a website history. Instead do as below:

How to update the website

1. Copy over parts of previous index.html, for now that is only this SwingPocWebReadme.txt
2. Amend commit in this local repo. no need to save the old website.
3. Delete gh-pages branch on https://github.com/swingmaster/swingmaster.github.io
4. push to remote  git push origin gh-pages
5. Verify https://swingmaster.github.io
